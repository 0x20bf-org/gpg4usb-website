Latest News - 2013-10-22

After a long time of silence we have some updates for you.

Domain move
Our domain has moved, now you will find us on http://gpg4usb.org, please update your links.

Stable Release now with gpg 1.4.15
We have bundled gnupg 1.4.15 with our stable 0.3.2 release. This gnupg version fixes a DoS Bug, see announcement. If you download version 0.3.2-1 now, it will contain the updated gpg binary.

FAQ & Known Bugs
We now have a FAQ and Known Bugs section on the website, please look there first if it answers your questions before emailing. Your emails are still welcome, but some questions should be explained after looking there.

MacOSX & unstbale builds
Frontlinedefenders were so kind to provide us with a mac mini, so we could develop a mac osx version of gpg4usb. Frontlinedefenders is a human rights organisation, which is providing trainings in communication security to human rights defenders.
So we have been working silently on a mac-version of gpg4usb, which was not as easy as we first thought. We exchanged our underlaying gpgme library to that one from kgpg, which needed modifications in our whole code base. But it is easier to maintain by us. This transition is not yet finished, but we decided that it starts to be sufficient to be used and tested by you. So from now on you could download unstable builds from http://gpg4usb.org/nightly/. We will update this directory whith more unstable builds as we move along to our next stable release.
Please send feedback about bugs you find, if they are not yet listed on the Known Bugs page.

Donations
Some people told us that they like gpg4usb and would want to donate. We are happy if you enjoy gpg4usb, and about your feedback. If you want to donate we would be happy if you support Frontlinedefenders, as they could do better things with the money, and, you know, they provided us with the mac mini :).
